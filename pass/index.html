pokemogukunnの、プログラミングコーザ(HTML)

HTMLタグの基本的な使い方とコピー機能

1. メタ情報やSEO関連のタグ

ページのメタ情報やSEOに関連するタグです。

<meta charset="UTF-8">
<meta name="description" content="ページの説明">
<meta name="keywords" content="HTML, CSS, JavaScript">
<link rel="stylesheet" href="styles.css">

2. スクリプトとスタイルのタグ

CSSスタイルやJavaScriptを埋め込むためのタグです。

<style>
    body { background-color: lightblue; }
</style>

<script>
    alert("ページが読み込まれました");
</script>

<script src="script.js"></script>

3. マルチメディア関連のタグ

音声や動画、その他のメディアを埋め込むためのタグです。

<audio controls>
    <source src="audio.mp3" type="audio/mpeg">
    お使いのブラウザはオーディオタグをサポートしていません。
</audio>

<video controls width="300">
    <source src="video.mp4" type="video/mp4">
    お使いのブラウザはビデオタグをサポートしていません。
</video>

<embed src="file.pdf" width="500" height="375">

<iframe src="https://example.com" width="600" height="400">
    このブラウザはiframeをサポートしていません。
</iframe>

4. セマンティックHTMLタグ

ページの意味を明確にするためのセマンティックなタグです。

<article>
    <h2>記事タイトル</h2>
    <p>この記事の内容。</p>
</article>

<aside>
    これは補足情報です。
</aside>

<header>
    これはヘッダーです。
</header>

<footer>
    これはフッターです。
</footer>

<nav>
    <ul>
        <li><a href="#home">ホーム</a></li>
        <li><a href="#about">概要</a></li>
    </ul>
</nav>

5. フォーム要素

ユーザーからの入力を受け取るためのフォーム要素です。

<form>
    <label>名前: <input type="text" name="name"></label><br>
    <label>パスワード: <input type="password" name="password"></label><br>
    <textarea name="message">ここにメッセージを入力</textarea><br>
    <select name="options">
        <option value="option1">オプション1</option>
        <option value="option2">オプション2</option>
    </select><br>
    <input type="submit" value="送信">
</form>

6. テーブル要素

表を作成するための要素です。

<table>
    <tr>
        <th>項目1</th>
        <th>項目2</th>
    </tr>
    <tr>
        <td>データ1</td>
        <td>データ2</td>
    </tr>
</table>

7. グラフィックと描画のタグ

グラフィックやベクター画像を描画するためのタグです。

<canvas id="myCanvas" width="200" height="100">
    このブラウザはcanvasをサポートしていません。
</canvas>

<svg width="100" height="100">
    <circle cx="50" cy="50" r="40" stroke="black" stroke-width="3" fill="red" />
</svg>
