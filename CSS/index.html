CSSの基本的な使い方とコピー機能

1. インラインスタイル

HTML要素に直接スタイルを適用する方法です。

<h1 style="color: blue;">こんにちは！</h1>

2. 内部スタイルシート

HTMLドキュメントの`<head>`タグ内にスタイルを定義します。

<style>
    h1 {
        color: red;
    }
</style>
<h1>こんにちは！</h1>

3. 外部スタイルシート

外部CSSファイルをHTMLにリンクしてスタイルを適用する方法です。

<link rel="stylesheet" href="styles.css">

4. クラスセレクタ

クラスセレクタを使って、複数の要素に同じスタイルを適用します。

<style>
    .highlight {
        color: green;
        font-weight: bold;
    }
</style>
<p class="highlight">このテキストは強調されています。</p>

5. IDセレクタ

IDセレクタを使って、特定の要素にスタイルを適用します。

<style>
    #unique {
        color: orange;
        text-align: center;
    }
</style>
<p id="unique">このテキストはIDセレクタでスタイルが適用されています。</p>

6. 要素セレクタ

要素の種類ごとにスタイルを適用します。

<style>
    p {
        color: purple;
        font-size: 18px;
    }
</style>
<p>この段落は要素セレクタでスタイルが適用されています。</p>

7. 擬似クラス

マウスホバーなど、特定の状態にある要素にスタイルを適用します。

<style>
    a:hover {
        color: red;
    }
</style>
<a href="#">リンクにカーソルを合わせてみてください</a>

8. フレックスボックス

レイアウトを作成するためのフレキシブルなボックスモデルです。

<style>
    .container {
        display: flex;
        justify-content: space-between;
    }
    .item {
        background-color: lightblue;
        padding: 10px;
        margin: 5px;
    }
</style>
<div class="container">
    <div class="item">アイテム1</div>
    <div class="item">アイテム2</div>
    <div class="item">アイテム3</div>
</div>

9. グリッドレイアウト

2次元のレイアウトを作成するためのCSSグリッドシステムです。

<style>
    .grid-container {
        display: grid;
        grid-template-columns: auto auto auto;
        gap: 10px;
    }
    .grid-item {
        background-color: lightgreen;
        padding: 20px;
        text-align: center;
    }
</style>
<div class="grid-container">
    <div class="grid-item">1</div>
    <div class="grid-item">2</div>
    <div class="grid-item">3</div>
    <div class="grid-item">4</div>
    <div class="grid-item">5</div>
    <div class="grid-item">6</div>
</div>

10. メディアクエリ

レスポンシブデザインを実現するために、画面サイズに応じてスタイルを変更します。

<style>
    body {
        background-color: white;
    }
    @media (max-width: 600px) {
        body {
            background-color: lightgrey;
        }
    }
</style>
<p>画面サイズを600px以下にすると背景色が変わります。</p>
