JavaScriptの基本的な使い方とコピー機能

1. 基本的なJavaScriptの埋め込み

JavaScriptをHTMLに直接埋め込む方法です。HTML内の`<script>`タグ内にJavaScriptコードを記述します。

<script>
    alert('Hello, World!');
</script>

2. 外部JavaScriptファイルを読み込む

外部のJavaScriptファイルをHTMLに読み込む方法です。JavaScriptファイルを再利用しやすくするために使われます。

<script src="script.js"></script>

3. イベントリスナーを使ったインタラクション

JavaScriptでイベントリスナーを使い、ユーザーの操作に応じて処理を行う方法です。

<button id="myButton">クリックして！</button>
<script>
    document.getElementById('myButton').addEventListener('click', function() {
        alert('ボタンがクリックされました！');
    });
</script>

4. DOM操作

JavaScriptを使ってHTML要素を動的に操作する方法です。

<div id="content">ここにテキストが表示されます</div>
<script>
    document.getElementById('content').textContent = 'JavaScriptでテキストを変更しました！';
</script>

5. タイマー機能（setTimeoutとsetInterval）

JavaScriptでタイマーを使って処理を遅延させる方法です。

<script>
    // 3秒後にアラートを表示
    setTimeout(function() {
        alert('3秒後に表示されました！');
    }, 3000);

    // 1秒ごとにカウントを表示
    let count = 0;
    setInterval(function() {
        console.log('カウント: ' + count);
        count++;
    }, 1000);
</script>

6. 条件分岐 (if/else)

条件に応じて異なる処理を行うための`if`や`else`文を使用します。

<script>
    let number = 10;

    if (number > 5) {
        console.log('数字は5より大きい');
    } else {
        console.log('数字は5以下です');
    }
</script>

7. 繰り返し処理 (for/while)

指定した回数だけ処理を繰り返すための`for`や`while`ループを使用します。

<script>
    // forループ
    for (let i = 0; i < 5; i++) {
        console.log('forループの回数: ' + i);
    }

    // whileループ
    let j = 0;
    while (j < 5) {
        console.log('whileループの回数: ' + j);
        j++;
    }
</script>

8. 関数の定義と呼び出し

JavaScriptで関数を定義し、それを呼び出す方法です。

<script>
    // 関数の定義
    function greet(name) {
        return 'こんにちは、' + name + 'さん！';
    }

    // 関数の呼び出し
    console.log(greet('太郎'));
</script>

9. 配列とオブジェクト

配列やオブジェクトを使ってデータを管理する方法です。

<script>
    // 配列
    let fruits = ['りんご', 'バナナ', 'ぶどう'];
    console.log(fruits[0]);  // りんご

    // オブジェクト
    let person = {
        name: '太郎',
        age: 30
    };
    console.log(person.name);  // 太郎
</script>
